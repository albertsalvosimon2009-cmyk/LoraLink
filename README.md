This project aims to build a pair of handheld, long-range text communication devices that operate entirely without cellular networks, Wi-Fi, or satellites. Using LoRa (Long Range) radio technology, these devices allow for private, point-to-point messaging in remote areas or emergency situations.

The device features a ruggedized enclosure, an OLED display for viewing messages, and a 3x4 T9 numeric keypad for tactile text input, reminiscent of classic mobile phones.

Core Features
Off-grid Communication: Operates on sub-GHz radio frequencies (915MHz/868MHz).

Low Power Consumption: Designed to run for days on a single 18650 Li-ion battery.

Tactile Interface: Physical keypad for reliable typing in any condition.

Privacy-Focused: No central server or third-party provider involved in the message transmission.

Hardware Stack (Planned)
Microcontroller: ESP32 (specifically the Lilygo TTGO T-Beam or Heltec LoRa 32).

Radio Module: Semtech SX1276 LoRa transceiver.

Display: 1.3" I2C OLED Screen.

Input: 4x3 Matrix Membrane or Plastic Keypad.

Enclosure: Custom 3D-printed rugged case.
